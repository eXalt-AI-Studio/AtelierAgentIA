| Pré-requis | Setup |
|:----------|:-----|
| Git | `git clone https://github.com/eXalt-AI-Studio/AtelierAgentIA.git` <br> `cd AtelierAgentIA` |
| Python 3.10+ | `python -m venv .ateliervenv` |
| Jupyter Notebook ou JupyterLab | `source .ateliervenv/bin/activate` (macOS/Linux) <br> `.ateliervenv\Scripts\activate` (Windows) |
| | `pip install -r requirements.txt` |
| | `cp .env.example .env` (éditer avec les clés APIs)|

**Configuration LLM** : compte [OpenRouter](https://openrouter.ai) → clé API → `OPENROUTER_API_KEY` dans `.env`
