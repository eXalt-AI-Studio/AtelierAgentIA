## Local

Python `3.12.x` pré-requis.

```bash
git clone https://github.com/eXalt-AI-Studio/AtelierAgentIA.git
cd AtelierAgentIA
python -m venv .ateliervenv
python -m pip install -r requirements.txt
```

Dupliquer `.env.example` en `.env`, renseigner `OPENROUTER_API_KEY`, puis ouvrir `atelier_agents_local.ipynb`.
